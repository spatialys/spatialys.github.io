Key accomplishments in GDAL/OGR

Maintainer of the Java bindings and their documentation.

Design and implementation of the OGR SQL SQLite dialect.

Design and implementation of multithreading in the computational part of the image warping.

Design and implementation of the GDAL_API_PROXY mechanism.

Design and implementation of the support for multiple geometry fields per vector layer, per RFC 41.

Design and implementation of the mechanism to expose GDAL datasets and raster bands as virtual memory mappings, per RFC 45

Unification of GDAL and OGR models, per RFC 46, for GDAL 2.0.

Reverse engineering of ESRI FileGDB format
